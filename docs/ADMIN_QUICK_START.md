# Admin Quick Start Guide

This guide helps you quickly set up and use the admin dashboard.

## Accessing Admin Panel

1. Navigate to `/admin` after logging in
2. Ensure your user has admin role in Firestore

## Key Admin Features

- **Games Management**: Configure game settings and points
- **XP & Tiers**: Customize tier system and rewards
- **Vouchers**: Create and manage discount vouchers
- **Events**: Manage events and registrations
- **Users**: View and manage user accounts
